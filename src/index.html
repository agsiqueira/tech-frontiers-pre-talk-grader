Obtain a UF NaviGator API Key

Go to the UF NaviGator portal:
https://api.ai.it.ufl.edu/ui

Log in with your UF credentials.

Click Create New Key in the upper-left area near the sidebar.

Fill out the form as follows:

  • Owned By: You
  • Team: navigator-toolkit
  • Key Name: Choose a descriptive name such as tf-debriefing
  • Models: granite-3.3-8b-instruct
  • Key Type: AI APIs

Important API Key Security Information

Save your API key in a safe location. You may not be able to view the full key again after leaving the page.

Never share your API key with anyone.
